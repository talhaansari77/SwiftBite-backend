# SwiftBite Backend 🍔

A full-stack food delivery app backend built with Node.js, Express, MongoDB and TypeScript.

## Tech Stack
- Node.js + Express
- TypeScript
- MongoDB + Mongoose
- JWT Authentication
- bcryptjs

## Features
- User authentication (register/login)
- Restaurant & menu management
- Order placement & tracking
- Protected API routes
